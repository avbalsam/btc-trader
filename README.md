# btc_algo
The btc_algo project aims to develop algorithms for trading crypto and fiat currencies.
## Project Goals
This overview of project goals in subject to change:
* BTC taker style arbitrage on US exchanges
* ETH (and other) crypto arbitrage on US exchanges and international exchanges
* Maker style trading on local exchanges
* Trading on international crypto exchanges and necessary currency swaps
* FX and crypto trading on local and international markets
